# Karanova Landing - Bun Edition 🚀

## Quick Start

```bash
# Development
.\bun.cmd dev

# Production Build
.\bun.cmd build

# Start Production Server
.\bun.cmd start
```

## Why Bun?

- **25x faster** installs than npm
- **Native TypeScript** support
- **Built-in bundler** and test runner
- **Drop-in Node.js replacement**
- **Production-ready** and stable

## Available Scripts

| Command | Description |
|---------|-------------|
| `.\bun.cmd dev` | Start development server on localhost:3000 |
| `.\bun.cmd build` | Create production build |
| `.\bun.cmd start` | Start production server |
| `.\bun.cmd lint` | Run ESLint checks |
| `.\bun.cmd install` | Install dependencies |
| `.\bun.cmd add <pkg>` | Add a new package |
| `.\bun.cmd remove <pkg>` | Remove a package |

## Bun Location

Your Bun installation: `D:\freeup\.bun\bin\bun.exe`

To use `bun` globally, add to your PATH:
1. System Properties → Environment Variables
2. Edit User PATH
3. Add: `D:\freeup\.bun\bin`
4. Restart terminal
