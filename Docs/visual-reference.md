## Color Palette

### Inova (Emerald/Teal)
- Headline: `from-emerald-200 via-teal-200 to-emerald-300`
- Indicator: `emerald-400` + glow
- Glow: `from-emerald-500/25 via-teal-500/20 to-cyan-500/15`

### TaskEase (Cyan/Blue)

## Animation Timeline

### Inova Hook
```
0.0s: Component enters viewport
0.1s: AI indicator fades in
0.3s: Headline slides in from left
0.5s: Subheadline slides in from left
0.7s: First progress dot animates
0.8s: Second progress dot animates
0.9s: Third progress dot animates
1.0s: Fourth progress dot animates
Loop: Pulsing indicator + breathing glow
```

### TaskEase Hook
```
0.0s: Component enters viewport
0.2s: Headline slides in from left
0.4s: Subheadline slides in from left
0.6s: Underline grows from right
1.6s: Shimmer effect starts
Loop: Shimmer travels + breathing glow
```
