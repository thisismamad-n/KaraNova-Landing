# Karanova Landing Page

Ultra-creative landing page for Karanova business management platform with bilingual support (English/Persian).

## Tech Stack

- **Framework**: Next.js 16 (App Router) with React Compiler
- **React**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, @number-flow/react
- **Icons**: Lucide React
- **Package Manager**: pnpm

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Run development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000)
